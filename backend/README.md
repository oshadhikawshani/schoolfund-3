# Backend (MERN)

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Create a `.env` file with your MongoDB URI:
   ```env
   MONGO_URI=your_mongodb_connection_string_here
   PORT=5000
   ```
3. Start the server:
   ```bash
   node index.js
   ```

## Tech Stack
- Node.js
- Express
- MongoDB (Mongoose)
- CORS
- dotenv
